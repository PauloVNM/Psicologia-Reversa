# 🍓 [NOT] A STRAWBERRY | Reality Check

> "As pessoas são como morangos: amargos, estragam rápido e, se tiverem sorte, são 'bonitos'."

Um projeto satírico desenvolvido para combater a positividade tóxica, o excesso de coaching e a ilusão de que o universo te deve um final feliz. 

## 🛠️ Tecnologias Utilizadas
A engenharia foi mantida simples, funcional e direta, sem frameworks inchados:
* **HTML5:** Estrutura semântica e pura.
* **CSS3:** Layout construído estritamente com **CSS Grid** (para particionamento estrutural) e **Flexbox** (para alinhamento interno).
* **JavaScript (Vanilla):** Lógica direta para validação de Regex no client-side e manipulação do DOM, impedindo o envio de formulários inúteis sem recarregar a página.

## ⚙️ Funcionalidades
1. **Gerador de Desmotivação:** Um motor de choque de realidade estático.
2. **Coleta de Culpados:** Formulário frontend para assinar seu atestado de incompetência psicológica.
3. **Layout Responsivo:** O abismo se adapta perfeitamente tanto a monitores ultrawide quanto à tela rachada do seu celular.
