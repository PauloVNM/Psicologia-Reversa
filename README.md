🍓 [NOT] A STRAWBERRY | Reality Check
"As pessoas são como morangos: amargos, estragam rápido e, se tiverem sorte, são 'bonitos'."

Um projeto satírico desenvolvido para combater a positividade tóxica, o excesso de coaching e a ilusão de que o universo te deve um final feliz.

🛠️ Tecnologias Utilizadas
A engenharia foi mantida simples, funcional e direta, sem frameworks inchados:

HTML5: Estrutura semântica e pura.

CSS3: Layout construído com CSS Grid e Flexbox para garantir que o caos seja organizado.

JavaScript (Vanilla): Lógica pura para manipulação do DOM e validação manual de formulários.

EmailJS API: Utilizada para o disparo de ofensas automatizadas diretamente para a caixa de entrada do usuário.

⚙️ Funcionalidades
Gerador de Desmotivação: Motor estático que entrega verdades inconvenientes via alerta.

Avaliação Psicológica (O Golpe): Formulário que promete um quiz, mas utiliza o EmailJS para enviar um e-mail confirmando que não ligamos para os seus problemas.

Validação Hostil: Sistema de validação de Regex que humilha o usuário caso ele não consiga digitar um e-mail corretamente.

Layout Responsivo: O abismo se adapta perfeitamente tanto a monitores ultrawide quanto à tela rachada do seu celular.

📖 Seções Estruturais (Burocracia Acadêmica)
Sobre o Projeto: Explicação do objetivo e do público-alvo exausto.

Problema que resolve: Resolve o meu problema (ganhar nota) enquanto menospreza o seu.

Metodologia: Engenharia baseada em Gemini, Ctrl+C e Ctrl+V.

Resultados Esperados: Obtenção de nota para aprovação na disciplina.
